使用 Python 进行数据分析之环境搭建

2017-12-25

  • 系统 Mac os 10.12.6

  • 安装 edm

下载 edm 安装器: https://www.enthought.com/products/edm/installers

  • 安装模块
% edm install numpy
% edm install pandas
% edm install ipython
% edm list # 查看安装了哪些模块
% edm shell # 进入 edm 环境
% ipython
% jupyter notebook

然后你还可以玩 Raku:

进入 edm shell 后:

jupyter notebook --generate-config
jupyter console --generate-config
