在命令行中导入 Maven 包

2017-04-06

mill 是一个新的 build 工具, 可以在命令行中导入 jar 包:

import $ivy.`com.alibaba:fastjson:1.2.12`
import com.alibaba.fastjson.{JSON, JSONObject}

val j = JSON.parseObject("{'a':123}")
println(j.get("a")) // 123
